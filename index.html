字段采集脚本怎么用
  1. 复制字段采集脚本，打开招聘网站的投递表单页。
  2. 按 F12 或 Ctrl + Shift + J，进入 Console 控制台。
  3. 粘贴脚本并回车；如果浏览器禁止粘贴，先在 Console 输入 allow pasting 后回车。
  4. 脚本会复制当前页面字段 JSON，回到这里粘贴到上方输入框。
  5. 点击本地匹配或 AI匹配，再重新生成自动填表脚本。
allow pasting 仍然失败： 打开 DevTools 的 Sources / 源代码 面板，进入 Snippets / 代码段，新建代码段后粘贴脚本，按 Ctrl + Enter 运行；或者直接把投递页的小标题一行一个手动粘贴到上方输入框。

适合处理“项目经历 / 代表项目 / 项目经验”“获奖 / 荣誉 / 竞赛经历”等标题不一致但含义相近的字段。AI 匹配会把当前简历和这些字段标题发送到你配置的 API。

推荐使用方式

更适合普通用户
  1. 点击 `准备投递`，系统会根据当前简历生成投递信息和脚本。
  2. 点击 `打开投递页面`，在招聘网站登录并进入申请表单。
  3. 把下面的 `一键填表书签` 拖到浏览器书签栏，之后在投递页点击它。
  4. 检查姓名、邮箱、电话、作品链接、求职信和附件，再手动提交。
